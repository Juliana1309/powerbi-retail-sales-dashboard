Descripción

Proyecto de análisis de ventas desarrollado en Power BI utilizando la base de datos Online Retail.

El objetivo fue analizar el comportamiento de las ventas, identificar clientes y productos destacados, evaluar la participación por país y estudiar tendencias temporales mediante indicadores de negocio.

1. Tecnologías Utilizadas:
Power BI
Power Query
DAX
Excel

2. KPIs Desarrollados
Ventas Totales
Clientes Únicos
Pedidos Totales
Ticket Promedio
Crecimiento Mensual (%)

3. Análisis Realizados
Top 10 Clientes por Ventas
Top 10 Productos por Ventas
Participación de Ventas por País
Tendencia Mensual de Ventas
Comparación contra Mes Anterior

4. Resultados
Más de 400.000 registros analizados.
4.372 clientes evaluados.
25.900 pedidos analizados.
